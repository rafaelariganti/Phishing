# Simulação de Phishing - Netflix

Repositório destinado à entrega da atividade prática da disciplina de **Segurança e Defesa Cibernética** [ministrada pelo professor Emerson Rocha.

## Sobre a Atividade
O objetivo deste trabalho foi desenvolver uma página que simule a tela de login legítima da **Netflix** para demonstrar, de forma prática, o funcionamento de um ataque de engenharia social do tipo **Phishing**. O sistema solicita informações do usuário (como e-mail e senha) e as armazena para posterior consulta e análise, ilustrando como criminosos capturam dados sensíveis.

>  **Aviso Ético:** Este projeto tem fins estritamente educacionais e acadêmicos para o estudo de defesa cibernética. O uso deste código para enganar indivíduos reais ou roubar credenciais é ilegal e antiético.

## O que foi entregue neste repositório:
- **Interface Clonada (Front-end):** Código da página falsa simulando visualmente a tela de login da Netflix.
- **Sistema de Captura (Back-end):** Código responsável por receber os dados inseridos pelo usuário e gravá-los no sistema.
- **Banco de Dados:** Estrutura utilizada para armazenar as credenciais capturadas na simulação.
